## Get started

### Configuration

```
TELEGRAM_BOT_TOKEN="YOUR_TELEGRAM_BOT_TOKEN"
OPENAI_API_KEY="YOUR_OPENAI_API_KEY"
DB_REDIS="YOUR_REDIS_URL"
```

```shell
git clone https://github.com/al002/llm_bot.git

cd llm_bot
```

copy `.env.example` to `.env` then change configuration.

### Run
```
cargo run
```
